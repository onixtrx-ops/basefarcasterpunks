Important before deploying:
1. Replace placeholders in public/.well-known/farcaster.json (accountAssociation.header, payload, signature)
   with values generated by Farcaster 'Generate domain manifest' tool or your signed manifest.
2. On Vercel set Environment Variables:
   - RELAYER_PRIVATE_KEY (0x...)
   - BASE_RPC_URL (https://mainnet.base.org or your RPC)
   - WEBHOOK_SECRET (random hex string)
   - NFT_CONTRACT (your token contract)
   - SEADROP_ADDRESS (SeaDrop contract if different)
   - OPTIONAL_MAX_PER_TX (3)

3. Deploy to Vercel and verify the manifest at:
   https://<your-vercel-domain>/.well-known/farcaster.json
4. Add the domain in Warpcast Developer -> Domains (enter domain without protocol), generate domain manifest and verify.

Security note: Do NOT commit private keys to GitHub. Use Vercel Environment Variables.
